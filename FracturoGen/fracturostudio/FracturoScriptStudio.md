# MANUEL DU PROTO-RUNESMITH

## FONCTIONNALITÉS PRINCIPALES

### 1. GÉNÉRATION DE CODES
- Sélectionnez une rune, un lieu, un effet, une version
- Choisissez un template de syntaxe
- Cliquez sur "Générer Code"

### 2. MUTATION
- **Mutation Aléatoire** : Change un élément au hasard
- **Ajouter Glitch** : Corrompt la syntaxe (caractères spéciaux)
- **Hybrider** : Mélange avec un autre code de l'historique

### 3. ANALYSE
- Stabilité ontologique (30-100%)
- Portée géographique
- Durée estimée
- Effets secondaires prédits

## PRINCIPES DE SÉCURITÉ

1. **Niveau de Danger** : Score calculé (rune × version)
   - <10 : Sûr
   - 10-30 : Risque modéré  
   - 30-50 : Danger élevé
   - >50 : Extrême danger (ne pas générer)

2. **Effets Secondaires Communs** :
   - Goût de fer métallique
   - Flashbacks mémoriels
   - Synesthésie temporaire
   - Sensibilité électromagnétique

3. **Protocoles** :
   - Un code par jour maximum
   - Toujours noter le hash
   - En cas de malaise : arrêt immédiat

## TEMPLATES PERSONNALISÉS

Créez vos propres syntaxes avec :
- `{rune}` : Rune choisie
- `{lieu}` : Lieu d'ancrage  
- `{effet}` : Effet désiré
- `{version}` : Numéro de version

Exemple : `DEBUG:{rune}_{version}@{lieu}#{effet}#END`
